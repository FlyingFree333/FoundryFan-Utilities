# FoundryFan-Utilities
A collection of custom actions for Monk's Active Tile Triggers.

Damage Token(s) (D&D5e only)
Region Behavior Control
Rotate Tile (fixes standard rotation reversal bug)
Store Roll Result (requires Monk's Tokenbar)
Trigger Random Tile
